Trabalhando com listas

Listas Ordenadas:

Ordem Numérica com o parametro type="1":

  1. Para inserir uma lista ordenada, primeiro devemos abrir a tag <ol>,
  2. Depois abrir a tag <li> para o primeiro item,
  3. E para cada item, uma tag <li>,
  4. E aqui mais uma <li>,
  5. Ao final fechar a tag </ol>.
